# Pwe Ecommerce

### Tecnologias Usadas

- **Node 16.19.0**
- **Yarn 1.19.19**
- **React 18.2.0**
- **TypeScirpt 4.9.3**
- **Styled Components 5.3.6**
- **Storybook 6.5.15**
- **Vite 4.0.0**

### Como usar

*OBS: Para baixar as dependências desse projeto é preciso ter o yarn instalado quaso não tenha use o comando **npm install -g yarn***.

#### Baixar dependências

Digite **yarn** para baixar as dependências do projeto

#### Storybook 

Digite **yarn storybook** para ver o componentes no storybook

#### Site montado

Digite **yarn dev** para ver o site completo
